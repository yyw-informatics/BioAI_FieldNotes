Technical Notes

Metric Definitions

Engagement: likes + replies + 2× reposts + 1.5× quotes, summed across deduplicated versions of the story.

Story fit: a 0-100 score from target-model matches, first-person usage cues, task/outcome terms, domain context, visual demo evidence from X media metadata, and penalties for announcement-only or meta posts.

Deduplication: stories are merged by shared external links first, then near-duplicate text. Generic X/Twitter status links are ignored as dedupe keys.
